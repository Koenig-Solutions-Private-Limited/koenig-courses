## What You Will Learn
- Core bridging protocols including STP and RSTP for loop prevention in Ethernet networks
- VLAN tagging and trunking configurations to segment traffic at the enterprise edge
- Bridge domain setup using modern tools like Linux Bridge or eBPF for virtualized environments
- Troubleshooting methodologies for common edge bridging failures, such as broadcast storms
- Integration techniques linking edge bridges to core routers and SDN controllers

## Who Should Attend
This course suits junior network administrators, IT support technicians with basic networking exposure, aspiring edge engineers, and telecom field operators seeking structured entry into bridging technologies—typically those with 6-12 months of hands-on experience.
