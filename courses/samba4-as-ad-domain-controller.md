## What You Will Learn
- Provision a new Active Directory domain using samba-tool and configure Samba4 as the primary domain controller
- Manage users, groups, organizational units, and computer accounts via command-line tools and RSAT
- Set up internal DNS integration with Samba's BIND9_DLZ backend for reliable name resolution
- Implement Kerberos authentication and secure replication across multi-site Samba AD deployments
- Join Windows and Linux clients to the domain, including Group Policy Object emulation with rsync
