## What You Will Learn
- Install and configure SCCM hierarchies, including primary and secondary sites with database replication
- Deploy operating systems via task sequences, PXE booting, and multicast for efficient imaging
- Distribute software packages and applications using collections, requirements, and supersedence rules
- Manage software updates through WSUS integration, automatic deployment rules, and update groups
- Implement compliance settings, configuration baselines, and endpoint protection policies for security
