## What You Will Learn
- Configure initial Junos OS settings and user authentication on EX Series switches
- Implement and troubleshoot VLANs, including trunking and flexible VLAN tagging
- Deploy Rapid Spanning Tree Protocol (RSTP) and Multiple Spanning Tree Protocol (MSTP) for loop prevention
- Set up and monitor link aggregation groups (LAGs) using IEEE 802.3ad
- Apply Layer 2 security measures such as MAC limiting, DHCP snooping, and dynamic ARP inspection
