## What You Will Learn
- Reverse engineer Android APKs using APKTool and Jadx to expose hardcoded secrets and weak cryptography.
- Conduct static analysis on iOS IPA files with tools like otool and class-dump to identify insecure data storage.
- Perform dynamic instrumentation via Frida and Objection for runtime manipulation and API interception.
- Bypass jailbreak/root detection and SSL pinning to simulate real-world attack vectors.
- Assess OWASP Mobile Top 10 risks, including insecure authentication and improper platform usage.

## Who Should Attend
