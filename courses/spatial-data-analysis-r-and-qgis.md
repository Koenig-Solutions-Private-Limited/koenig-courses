## What You Will Learn

- Conducting spatial joins, overlays, and proximity analysis to uncover patterns in geographic datasets
- Building statistical models for spatial autocorrelation and hot-spot detection with R's spdep and SpatialEco libraries
- Automating geospatial workflows by integrating R scripts directly into QGIS via processing scripts
- Generating interactive maps and 3D visualizations for web publication using QGIS layouts and R's leaflet and tmap
