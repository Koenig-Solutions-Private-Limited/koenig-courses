## What You Will Learn

- Set up application stream proxies for protocols like HTTP, HTTPS, MAPI/Exchange, and CIFS file sharing
- Install, license, and perform initial configuration of SteelHead appliances in branch and data center environments
- Monitor performance metrics and generate reports using Riverbed Central Management Console for proactive issue resolution
