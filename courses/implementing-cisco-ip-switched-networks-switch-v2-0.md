## What You Will Learn
- Implement VLANs, VTP, and trunking to segment traffic efficiently across switched environments
- Configure Spanning Tree Protocol enhancements like RSTP, MSTP, and PortFast for loop prevention and fast convergence
- Set up EtherChannel bundles with LACP or PAgP for increased bandwidth and redundancy
- Deploy first-hop redundancy protocols including HSRP, VRRP, and GLBP for gateway failover
- Design multilayer switching architectures with OSPF and EIGRP integration for inter-VLAN routing
