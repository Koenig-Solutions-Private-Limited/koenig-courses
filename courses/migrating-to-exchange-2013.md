# Migrating to Exchange 2013 — Training & Certification Guide

## What You Will Learn
- Assess current Exchange environments and plan migration paths, including prerequisites for Exchange 2013 Server Roles.
- Configure co-existence between Exchange 2010/2007 and 2013, managing free/busy sharing and authentication.
- Execute mailbox moves using New-MoveRequest cmdlets and handle large-scale database portability.
- Deploy Database Availability Groups (DAGs) with improved witness servers and continuous replication.
- Perform post-migration cleanup, including legacy server decommissioning and OWA/ActiveSync validation.

## Who Should Attend
Systems administrators experienced with Exchange Server 2010 or earlier, messaging engineers planning upgrades, and IT consultants handling enterprise migrations. Requires solid Active Directory knowledge and 2+ years in email administration.
