## What You Will Learn
- Core NFV components including NFVi, VNFs, and orchestration layers in Cisco environments
- VNF lifecycle management: onboarding, instantiation, scaling, and termination using ETSI MANO standards
- Cisco-specific NFV platforms like Cisco NFV Infrastructure Software and integration with OpenStack
- Performance optimization techniques for virtualized EPC and IMS functions
- Troubleshooting NFV deployments, including fault isolation in multi-tenant setups
