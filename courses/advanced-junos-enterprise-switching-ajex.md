## What You Will Learn
- Configure and troubleshoot Virtual Chassis for stacking multiple EX Series switches into a single logical device
- Implement spanning-tree protocols including MSTP and VSTP for loop prevention in large-scale Layer 2 domains
- Design and apply QoS policies using BA classifiers, queues, and schedulers to prioritize critical traffic
- Deploy advanced security mechanisms like MAC limiting, storm control, and firewall filters on switch interfaces
- Utilize Junos troubleshooting tools such as RPL scripts and real-time monitoring for rapid issue resolution
