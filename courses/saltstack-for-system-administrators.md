## What You Will Learn
- Install and configure Salt master, minions, and proxies for multi-OS environments including Linux, Windows, and cloud instances
- Define and apply SLS state files to manage packages, services, files, and users idempotently across fleets
- Leverage Pillars for external data injection and Grains for inventory-based targeting without hardcoded values
- Perform ad-hoc remote execution with matchers, globbing, and compound targeting for precise control
- Build event-driven reactors and orchestration jobs to automate workflows like zero-downtime deployments
