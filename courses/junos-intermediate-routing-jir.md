# Junos Intermediate Routing (JIR) — Training & Certification Guide

The [Junos Intermediate Routing (JIR)](https://www.koenig-solutions.com/jir-juniper?utm_source=github&utm_medium=backlink&utm_campaign=github-publisher) course delves into advanced routing protocols on Juniper Networks devices using Junos OS. It equips network engineers and administrators with skills to configure, monitor, and troubleshoot OSPF, IS-IS, and BGP in enterprise and service provider environments. Ideal for those with basic Junos experience, this training addresses the growing need for robust routing in multi-domain networks, hybrid clouds, and secure data centers where protocol optimization ensures scalability and reliability amid rising traffic demands.

## What You Will Learn
- Configure and verify OSPF areas, including stub and NSSA types for efficient intra-domain routing
- Implement IS-IS levels and metrics to support large-scale link-state routing hierarchies
- Establish BGP peering sessions with iBGP and eBGP, including route reflection for scalability
- Design routing policies for route filtering, redistribution, and preference manipulation across protocols
- Troubleshoot routing issues using Junos tools like `show route`, `traceroute`, and protocol-specific diagnostics

## Who Should Attend
Network engineers, systems administrators, and pre-sales engineers with foundational Junos OS and routing knowledge (e.g., JNCIA-Junos certified). Suited for those transitioning to intermediate roles in service provider or enterprise environments handling complex topologies.
