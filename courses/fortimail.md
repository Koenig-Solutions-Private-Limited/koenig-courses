## What You Will Learn
- Deploy and initial configure FortiMail appliances in gateway, server, and transparent modes
- Implement anti-spam profiles using machine learning classifiers and reputation-based filtering
- Set up data loss prevention (DLP) policies with fingerprinting and content disarmament
- Configure antivirus scanning engines and quarantine management for inbound/outbound traffic
- Establish high availability clusters and integrate FortiMail with FortiGate for unified security

## Who Should Attend
