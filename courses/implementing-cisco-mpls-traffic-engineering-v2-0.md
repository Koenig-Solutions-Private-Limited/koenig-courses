## What You Will Learn
- Core MPLS TE concepts, including link attributes like bandwidth and affinities for path selection
- RSVP-TE protocol operations for establishing and maintaining TE label-switched paths
- Explicit route objects (ERO) and recorded route objects (RRO) for precise path computation and verification
- Traffic forwarding along TE tunnels, with load balancing and make-before-break procedures
- Fast reroute (FRR) mechanisms and dynamic path adjustments for high availability
