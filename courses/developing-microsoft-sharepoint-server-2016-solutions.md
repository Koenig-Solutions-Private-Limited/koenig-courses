## What You Will Learn
- Design and develop SharePoint apps using the app model, including provider-hosted and SharePoint-hosted approaches
- Implement client-side rendering with JavaScript Object Model (JSOM) and REST APIs for dynamic user interfaces

- Create and deploy custom SharePoint solutions with features, lists, and event receivers
- Customize workflows and integrate Business Connectivity Services (BCS) for external data access
