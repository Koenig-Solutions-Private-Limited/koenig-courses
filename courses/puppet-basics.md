## What You Will Learn
- Install and configure Puppet Open Source master and agent nodes on Linux/Unix systems
- Author Hiera data structures for dynamic node classification and fact-based decisions
- Define and manage resources like packages, files, users, and services using DSL manifests
- Develop reusable Puppet modules with dependencies and forge integration
- Troubleshoot common issues with Facter facts, catalog compilation, and agent runs
