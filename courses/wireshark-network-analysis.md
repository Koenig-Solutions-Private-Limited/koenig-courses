## What You Will Learn
- Deploy Wireshark for live packet capture on wired, wireless, and VPN interfaces, configuring promiscuous mode and ring buffers for high-volume traffic.
- Craft and apply display filters using BPF syntax to isolate TCP streams, UDP flows, and application-layer payloads like HTTP/2 or DNS queries.
- Dissect core protocols including Ethernet, IP, ICMP, ARP, and TLS, identifying fragmentation, retransmissions, and sequence anomalies.
- Perform statistical analysis with IO graphs, protocol hierarchies, and endpoint trackers to pinpoint bottlenecks and unusual patterns.
- Conduct advanced troubleshooting, such as VoIP call flows, wireless decryption, and export object reconstruction for malware forensics.
