## What You Will Learn
- Install and configure Active Directory Domain Services (AD DS) including domain and forest functionals levels
- Manage user accounts, groups, organizational units, and computer objects using Active Directory Administrative Center
- Implement Group Policy Objects (GPOs) for enforcing security settings, software deployment, and compliance
- Deploy and configure Active Directory Federation Services (AD FS) for single sign-on across hybrid environments
- Monitor AD health, perform backup/restore operations, and troubleshoot replication issues with tools like repadmin
