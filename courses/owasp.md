## What You Will Learn
- Mapping and exploiting vulnerabilities from the OWASP Top 10, such as injection flaws and broken access control
- Deploying OWASP ZAP for automated dynamic application security testing (DAST) scans
- Conducting manual security tests using OWASP Testing Guide phases, from configuration to reporting
- Implementing secure coding practices to mitigate cross-site scripting (XSS) and SQL injection risks
- Integrating OWASP principles into CI/CD pipelines for continuous security validation

## Who Should Attend
