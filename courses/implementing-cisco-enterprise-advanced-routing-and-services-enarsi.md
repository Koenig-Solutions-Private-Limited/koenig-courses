## What You Will Learn
- Implement and verify OSPFv2 and OSPFv3 for multi-area IPv4/IPv6 routing in enterprise topologies
- Configure eBGP and iBGP peering, including route maps and prefix lists for policy-based routing
- Deploy EIGRP named mode with wide metrics, stub routing, and authentication mechanisms
- Set up VRF-lite for route isolation and DMVPN Phase 3 for scalable hub-and-spoke VPNs
- Troubleshoot infrastructure services like NTP, NAT, syslog, SNMPv3, and IP SLA monitoring

## Who Should Attend
